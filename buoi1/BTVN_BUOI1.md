# BÀI TẬP PYTHON

## Bài 1: Python là ngôn ngữ thông dịch hay biên dịch? Tại sao?

Python là **ngôn ngữ thông dịch**. Chương trình được thực thi thông qua trình thông dịch nên không cần biên dịch toàn bộ trước khi chạy. Điều này giúp Python dễ học, dễ sửa và chạy trên nhiều hệ điều hành.

## Bài 2
### 1. Các kiểu dữ liệu trong Python
- int (số nguyên)
- float (số thực)
- complex (số phức)
- str (chuỗi)
- list (danh sách)
- tuple (bộ dữ liệu)
- set (tập hợp)
- dict (từ điển)
- bool (đúng hoặc sai)

### 2. Các toán tử trong Python
- Toán tử số học
- Toán tử so sánh
- Toán tử gán
- Toán tử logic
- Toán tử thành viên
- Toán tử định danh

### 3. Mệnh đề điều kiện và vòng lặp
 #### Điều kiện
- if
- if...else
- if...elif...else

 #### Vòng lặp
- for
- while
- break
- continue
- pass

### 4. Kiểu dữ liệu True, False
`True` và `False` thuộc kiểu dữ liệu **bool**. Chúng biểu diễn hai giá trị đúng và sai, thường được dùng trong các biểu thức so sánh và câu lệnh điều kiện.
